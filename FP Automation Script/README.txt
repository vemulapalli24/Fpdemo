Extract shared zip folder

---- Edit config.properties before running sanity test ----

1. Open config.properties file 
2. Set "environment" variable to "dzeeSiteDemoUrl" | "demoUrl" | "fpDevUrl" | "fpQaUrl" | "fpQaEnvestnetUrl" value
3. Set "pdfEmail" variable value as your email address


---- Steps for running Automation suite Jar ----

1. Open command prompt and traverse to the extracted folder
2. Run the testsuite jar using "java -jar testsuite.jar" command


---- Test reports ----

The test results can be found in "test report" folder
